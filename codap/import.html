CSV Column Import

Upload a CSV file, then select a variable from the drop down that appears. To import this data into your CODAP sampler, copy the output text or click the copy button. On the CODAP sampler, click the button with the three dots under the device and paste the text into the window that pops up. This will import all the data into your sampler.









Important note: CODAP will not import data that has commas in the entries themselves correctly. Even if set up as a list like "New York, NY", "Boston, MA", CODAP will read these into the sampler as four mixer balls rather than two. If you want to import variables to CODAP from a data set where some of other variables may have commas, this tool will still load the other variables appropriately.
